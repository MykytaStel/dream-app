# Project Docs

This folder contains the main product and technical documentation for **Kaleidoscope of Dreams**.

## Included Files

- `../CLAUDE.md` — top-level project context and AI guidance
- `SKILLS.md` — how an AI assistant should collaborate on the project
- `PRODUCT.md` — product vision, target users, value proposition
- `ARCHITECTURE.md` — architecture direction and technical boundaries
- `DESIGN.md` — design principles and UX direction
- `MVP-ROADMAP.md` — phased product roadmap
- `FEATURE-BACKLOG.md` — prioritized feature backlog
- `MONETIZATION.md` — monetization strategy
- `GO-TO-MARKET.md` — launch and positioning strategy
- `ANALYTICS.md` — privacy-conscious analytics plan
- `TECH-STACK.md` — recommended technical stack

## Why This Structure

- `CLAUDE.md` at repo root gives top-level project context.
- `docs/` keeps all secondary reference material organized.
- this structure scales well if more documents are added later.
