# Release 0.0.24

## Theme

Import and restore foundation, with shell polish.

## Main outcomes

- define the first local import and restore path with preview-first trust
- continue turning the shell into a stronger branded product surface
- improve the bottom tab bar and active states without hurting clarity
- add controlled depth, glow, and motion where they reinforce hierarchy

## Small new feature

- add an import preview step before any local restore is committed

## Scope lock

Do not turn this into:

- cloud sync
- accounts
- subscriptions
- decorative motion that slows down core flows

Keep the visual pass focused on:

- bottom tab shell
- active controls
- retrieval surfaces
- subtle success feedback after capture

## Definition of done

- runtime and mobile build metadata point to `0.0.24`
- the bottom bar feels more intentional and branded
- active controls are easier to notice without reading as noise
- shell depth and accent color feel consistent across key screens
- import and restore can be implemented next without revisiting the shell again
